== Description ==

Custom reviews for Woocommerce plugin helps you to add custom reviews and ratings to your woocommerce products from wp admin dashboard.

== Installation ==

1. Upload the `custom-ratings-reviews-woocommerce` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.
